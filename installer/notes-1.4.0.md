Clarion Debugger 1.4.0

What's new since 1.3.1:

- New source editor (AvalonEdit) — the source view is now a proper code editor:
  - Clarion syntax colorizing (keywords, strings, comments, numbers, column-1 labels);
  - breakpoint margin/gutter and current-line highlight;
  - Go to line (Ctrl+G).

Download:
- ClarionDebuggerSetup-1.4.0.exe - installer (per-user, no admin)
- ClarionDebugger-1.4.0-portable-win-x86.zip - portable single .exe (unzip & run)

Self-contained (.NET 8 runtime bundled - nothing to install). 32-bit, runs on 64-bit Windows.
